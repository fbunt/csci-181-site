Percolation CSCI 232 - Data Structures

Assignment Description

We were tasked with developing an implementation of a percolation system. If a path exists from the top to the bottom through adjacent open nodes, then the system "percolates". The implementation had to allow for a variable grid size and be able to dynamically open nodes. It also had to meet several memory and running-time requirements.

My Solution

For my solution, I was able to identify and implement a solution for the "backfill" issue which had not been discussed in class.

Deliverables

  • Percolation implementation
  • A class for calculating stats for the percolation system
  • A project readme answering assignment questions

Takeaways

  • A much better understanding of the UnionFind data structure
  • An understanding of the Java memory model
