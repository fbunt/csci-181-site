Moon Lander PHSX 330 - Computational Physics

Assignment Description

We had to develop a program that modeled the Earth-Moon system along with an orbiting lander using the standard gravitational equations. We then had to find a "boost" velocity and trajectory that would cause lander to leave Earth's orbit and "land" on the Moon.

My Solution

I was able implement a program that correctly modeled the system and allowed the user to supply boost parameters. Through trial and error I was able to find boost parameters that resulted in a landing.

Deliverables

  • An orbital simulation program
  • Boost parameters that resulted in a landing
  • Boost parameters that resulted in an "interesting" landing
  • A project report

Takeaways

  • Knowledge of natural units and how to use them in a simulation
  • A better understanding of how to break up vectorized differential equations for computational modeling.
