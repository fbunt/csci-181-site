GeorgeDraw CSCI 136 - Fund. of CS II

Assignment Description

We were given the GeorgeDraw program and had to make additions to it while making sure that nothing broke the unit tests. The changes could include a new feature, adding more colors or a new shape, adding more unit tests or even fixing any issues/bugs.

My Solution

My solution included several changes and additions to the program. I added a new circle shape with complete with unit test and several new colors to the selection. I also refactored and fixed the square shape so that it could grow along any diagonal. Finally, I refactored the kock segment and the new circle so that they could be drawn with a variable amount of detail and placed a control on the UI so that a user could set the amount of detail.

Deliverables

  • The altered GeorgeDraw program
  • A writeup summarizing my changes

Takeaways

  • More experience with Java's Swing and AWT APIs
  • Learned to cap koch segment's detail at 5. 10 -> 2 GB for one line
