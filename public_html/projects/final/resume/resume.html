Resume CSCI 181 - Web Design

Assignment Description

For this assignment we had to create and publish a resume using HTML for the markup. It had to be professional looking and based on a resume style we found online. We also had to use a certain number of HTML elements from a list in our book.

My Solution

The style that I found and decided to use required CSS so I HTML and CSS for my submission. I was able to get fairly close to the original style through a lot of experimentation.

Deliverables

  • A link to the published resume page
  • A scan of the HTML code with elements from the list circled

Takeaways

  • Learned how to use CSS
  • Gained more experience with HTML
