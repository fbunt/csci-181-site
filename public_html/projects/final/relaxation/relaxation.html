Relaxation Solver PHSX 330: Computational Physics

Assignment Description

For this assignment, we had to write a program to solve a system of equations with initial conditions using an iterative relaxation method.

My Solution

My solution allowed the user to choose between two different relaxation methods and to specify how many iterations to carry out on the system.

Deliverables

  • Relaxation solver
  • Assignment Report

Takeaways

  • A much better understanding of Python's 3D plotting library
